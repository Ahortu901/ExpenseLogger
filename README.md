# ExpenseLogger

A native iOS expense logging application built using **Swift** and **SwiftUI** in **Xcode**.

---

## Features
- Take photos of receipts using the device camera
- Select receipt images from the photo library
- Automatically save a timestamp for each receipt
- Manually enter a claim amount per receipt
- Display a running total of all expenses
- Local data persistence (data remains after app restart)

---

## Tech Stack
- Swift
- SwiftUI
- SwiftData
- Xcode

---

## Requirements
- Xcode 15 or later
- iOS 17.0+
